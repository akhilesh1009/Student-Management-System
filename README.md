
#  Student Management Application

## Description of Application

The student management application was created for ABC College, which is a local education provider that specialises in 
software development, information management, and mobile application development training. This program provides features to 
capture student information, search for students, update student details, delete student records, and print student reports.


## Minimum System and Hardware Requirments

System Requirments - 
	- Operating System - Windows 8. 10 or 11.
	- Java Development Kit (JDK) version 9 or later.

Hardware Requirments - 
	- CPU - 1.8 GHz or faster processor.
	- RAM - 2GB, 8 GB of RAM recommended 
	- Storage - 800MB up to 210 GB
	
		          
## How to run Application

1. Make sure that you have NetBeans 8 or later installed (if not install it first).

2. Download the application that will be a zipped folder from GitHub or VC Learn.

3. Extract the folder and import it to NetBeans.

4. Navigate to Question 1 package, select the Main class, right click then select "Run File".

5. Check to see if the program runs correctly.

# HangMan Game 

## Description of Application

This application is a hangman game the objective is to guess a word related to colours. 
It allows the user to guess the colour by either guessing a letter or a full word. if a
guess is incorrect the hangman diagram will display the user will lose of of the 6 available 
attempts.
